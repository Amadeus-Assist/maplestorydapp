To run this frontend, first run
```
npm i
```
Then run 
```
npm rum serve
```
Then open 
```
http://127.0.0.1:8080/
```
to play the game.


Debug:
```
npm run build
```
to generate `main.js` in the `dist/` directory and it will watch the change of our `script/main.js`
Then run 
```
npm rum serve
```
Then open 
```
http://127.0.0.1:8080/
```
to play the game.
